Attach Scripts to your individual posts.

== Description ==

With this plugin, you can add custom CSS and JS files to your posts/pages. This allows you to add specific styles and scripts to your posts and pages without having to change your theme files.

A meta box will appear while editing a post or page that allows your to input and upload scripts.

Enjoy!

== Installation ==

To Install:

1. Download and unzip the contents of `postpage-headers.zip`
1. Upload the folder 'cat-template' to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Select script files to add when editing a post
